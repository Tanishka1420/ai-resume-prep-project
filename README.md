# AI Resume + Interview Prep Platform

## 🚀 About Project
This is a frontend web application that simulates an AI-powered resume analyzer.  
Users can upload their resume and get instant interview questions and feedback.

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## ⚡ Features
- Resume file upload UI
- AI-like analysis simulation
- Interview questions generator
- Clean dashboard UI
- Responsive design

## 📸 Project Flow
Upload Resume → Click Button → Get AI Analysis Output

## 💡 Future Improvements
- Real AI integration (OpenAI API)
- Backend with Node.js
- PDF parsing for resume
- User authentication
